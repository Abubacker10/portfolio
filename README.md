# Portfolio Website

This is a static portfolio website built with HTML, CSS, and JavaScript.

## How to Run

### Option 1: Open Directly (Simplest)
You can simply double-click the `index.html` file in your file explorer, or run the following command in your terminal:

```powershell
start index.html
```

### Option 2: Local Server (Recommended)
For the best experience (and to ensure all assets load correctly without browser security restrictions), run a local server.

**Using Python:**
```bash
python -m http.server
```
Then open `http://localhost:8000` in your browser.

**Using Node.js (npx):**
```bash
npx serve .
```

## Customization
-   **Content**: Edit `index.html` to update your text, links, and projects.
-   **Styles**: Modify `style.css` to change colors, fonts, or layout.
-   **Logic**: Update `script.js` for new interactive features.
